- **Base64 Operations**

- **URL Encoding**
  - URL encode text
  - URL decode text
  
- **Case Transformations**
  - UPPERCASE
  - lowercase
  - Title Case
  - camelCase
  - snake_case
  - kebab-case
  
- **Clipboard Operations**
  - Copy result to clipboard instantly
  - Paste from clipboard
  - Clear text
  
- **Keyboard Shortcuts**
  - `Ctrl+Alt+C` - Copy to clipboard
  - `Ctrl+Alt+V` - Paste from clipboard
